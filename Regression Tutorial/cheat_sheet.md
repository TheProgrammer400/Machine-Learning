# When To use What?

| Situation                             | Use                        |
| ------------------------------------- | -------------------------- |
| One feature, straight trend           | Linear Regression          |
| Multiple features, straight trend     | Multiple Linear Regression |
| Smooth curve                          | Polynomial Regression      |
| Weird nonlinear patterns              | Decision Tree              |
| Decision Tree overfits                | Random Forest              |
| Want strong baseline for tabular data | Random Forest              |
